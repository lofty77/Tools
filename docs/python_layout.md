### 主程序
我们常常看到的主程序形如 
```
if __name__ == "__main__": 
    pass
```
那么它的作用是什么呢？
- `__name__` 系统变量用来表示该模块是被导入的还是被直接执行
- 如果模块是被导入的，`__name__`的值为模块的名字
- 如果模块是被直接执行，`__name__`的值就为`'__main__'`
- 所以这个主程序就只有在被直接执行的时候会运行，而在被导入的时候避免被执行
